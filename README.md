# Calculadora


El codigo presentado implementa una calculadora funcional en una pagina web que utiliza HTML, CSS y JavaScript. La estructura HTML define la interfas incluyendo una pantalla para mostrar los resultados ademas de un conjunto de botones organizados en una cuadricula. La hoja de estilos en CSS mejora la apariencia proporcionando un dise√±o limpio, responsivo y centrado.

El java script agrega interactividad con la calculadora. Los botones estan enlazados mediante un evento de clic que identifica el contenido del boton pulsado. Tambien gestionaa operaciones basicas, el boton de borrar, la funcion de "igual" ( que evalua expreciones matematicas usando eval), y la liempiza de pantalla.
